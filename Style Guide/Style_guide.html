Interactive Elements

Navigation Button

CSS selector:

.button-nav

Sample code:

<a href="#" class="button">Default Button</a>

Rendered element:

Default Button

In-text link

CSS selector:

a

Sample code:

<a href="">This is a link.</a>

Rendered element:

This is a link.

CSS selector:

.button.submit
.button.reset

Sample code:

<input type="submit" class="button reset" value="Clear all">

Rendered element:
