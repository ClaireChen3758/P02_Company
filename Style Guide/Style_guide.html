Interactive Elements

Navigation Button

CSS selector:

.button-nav

Sample code:

<a href="#" class="button">Default Button</a>

Rendered element:

Default Button

In-text link

CSS selector:

a

Sample code:

<a href="">This is a link.</a>

Rendered element:

This is a link.

Submit/Reset Buttons

CSS selector:

.button.submit
.button.reset

Sample code:

<input type="submit" class="button submit" value="Submit form">
<input type="submit" class="button reset" value="Clear all">

Rendered element:

Text Input Field

CSS selector:

.textfield-label
.textfield

Sample code:

<label class="textfield-label" for="text-input-field">Name</label>
<input type="text" class="textfield" placeholder="Superman" id="input-first-name">

Rendered element:

Paragraphs

CSS selector:

p

Sample code:

<p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Phasellus et mollis tortor. Nunc porta augue ac quam lobortis, in porta risus varius.</p>

Rendered element:

Lorem ipsum dolor sit amet, consectetur adipiscing elit. Phasellus et mollis tortor. Nunc porta augue ac quam lobortis, in porta risus varius.
